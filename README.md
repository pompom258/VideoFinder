# VideoFinder

ローカルマシンの動画ファイルをスキャンし、メタデータ（再生時間）の管理やサムネイル表示、検索、再生を可能にする Web アプリケーションです。

## 概要

指定したディレクトリ配下の動画ファイルをスキャンし、動画の長さ（再生時間）を抽出して SQLite データベースで管理します。

Web ブラウザから動画の一覧を閲覧、検索できるほか、静止画およびアニメーション GIF によるサムネイル表示、ローカルアプリケーションでの直接再生といった機能を備えています。

## アーキテクチャ

```
VideoFinder/
├── src/
│   ├── app.ts          # アプリケーションエントリーポイント
│   ├── routes/         # ルーティング定義（API および ビュー）
│   ├── model/          # データアクセス層およびビジネスロジック
│   │   ├── repositories/ # データベースアクセス（SQLite3）
│   │   ├── services/     # 外部サービス連携（ffmpeg 等）
│   │   └── utils/        # 共通ユーティリティ
│   ├── entities/       # データモデル・型定義
│   ├── config/         # 定数・環境設定
│   └── events/         # クライアントサイド用スクリプト（イベントハンドリング）
├── public/             # 静的リソース（CSS, デフォルト画像等）
└── package.json        # 依存関係定義
```

### 主要コンポーネント

| コンポーネント | 説明 |
|--------------|------|
| `app.ts` | Express サーバーの初期化、静的ファイルの提供、ミドルウェア設定 |
| `routes/` | REST API (`/api`) および フロントエンド用 HTML の動的生成 |
| `model/` | SQLite データベースへのアクセスおよび ffmpeg を使用したメタデータ抽出、サムネイル生成 |
| `events/` | ブラウザ上で動作する検索、スキャン、再生等のイベント処理ロジック（TypeScript で記述） |

## 技術スタック

- **言語:** TypeScript (v5.5+)
- **フレームワーク:** Express.js
- **データベース:** SQLite3
- **動画処理:** fluent-ffmpeg (ffmpeg に依存)
- **HTTP クライアント:** Axios (内部 API 通信に使用)

## 主な機能

- **動画スキャン:** 指定したディレクトリ内の動画ファイルを再帰的に検索し、情報を DB に登録
- **メタデータ抽出:** 動画の長さ（再生時間）を自動抽出
- **サムネイル生成:**
  - スキャン時に静止画（PNG）サムネイルを生成
  - オンデマンドでアニメーション GIF サムネイルを生成
- **検索・フィルタリング:** キーワードによる動画ファイルの検索
- **ローカル再生:** ブラウザからの操作で、ローカルマシンの規定のアプリを使用して動画を再生

## API エンドポイント

| メソッド | エンドポイント | 説明 |
|---------|--------------|------|
| POST | `/api/scan` | ディレクトリをスキャンし、DB とサムネイルを初期化・更新 |
| GET | `/api/videos` | 動画一覧を取得（`keyword` クエリによる検索をサポート） |
| GET | `/api/play` | `videoId` で指定した動画をローカルで再生 |
| GET | `/api/getGif` | `videoId` に対応するアニメーション GIF を生成・取得 |

## ビルド・実行方法

※ 実行には `ffmpeg` および `ffprobe` がシステムパスに通っている必要があります。

```bash
# 依存関係インストール
npm install

# ビルドおよび実行（npx tsc 実行後に node で起動）
npm start
```

## 開発の背景

このプロジェクトは、ローカル上の動画ファイルの管理、検索を補助すると共に、以下の技術習得を目的として開発しました。

- TypeScript を用いたサーバーサイドおよびクライアントサイドの統合管理
- fluent-ffmpeg を使用したメディアファイル操作の実装
- SQLite3 を活用した軽量なメタデータ永続化
- Express.js によるシンプルな REST API と動的 HTML 生成の組み合わせ
